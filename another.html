Эта страница расположена по адресу /umodal/another.html

Чтобы загрузить в модальное окно содержимое другой страницы, необходимо указать её адрес в атрибуте umodal-src кликабельного элемента.

Из страницы автоматически отфильтруется всё лишнее и будет загружено только то содержимое, которое находится между тегами <body>

Также можно загрузить содержимое из определённого блока другой страницы, для этого есть специальный атрибут umodal-content В значение которого нужно указать любой существующий блок:

  • по классу: umodal-content=".class"
  • или ID: umodal-content="#id"
